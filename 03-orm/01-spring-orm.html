Spring ORM

Spring ORM

Spring ORM offre un support de :

  • Hibernate
  • Java Data Objects (JDO)
  • Java Persistence API (JPA)

Ce cours est centré sur le support de JPA.

Dépendance Maven :

<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-orm</artifactId>
</dependency>
